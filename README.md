## Search Engine for Shoes
단종된 신발과 비슷한 신발을 찾기 위해 네이버 이미지 검색(쇼핑렌즈)를 사용하던 중, 신발 검색만 제공하는 서비스를 만든다면 내가 더 잘할 수 있지 않을까? 하는 생각으로 시작했습니다.<br>
사용자가 신발 이미지를 입력하면 유사한 신발과 쇼핑몰을 찾아주는 신발 검색 엔진 서비스입니다.<br>
